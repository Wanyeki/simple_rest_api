# Rest api
## how to set up
1. clone the app locally
   ### dependancies
   1. ruby installed
   1. rails installed
1. cd to the downloaded folder using terminal
   ```bash
   $cd path/to/folder
   ```
1. run bundle install to install all dependancies
   ```bash
    $bundle install
   ```
 1. once done start the server
    ```bash
     $rail s
   ```
